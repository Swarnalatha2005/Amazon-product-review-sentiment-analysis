# Amazon-product-review-sentiment-analysis

## 📌 Project Overview
A machine learning project that classifies Amazon product reviews into
**Positive**, **Negative**, or **Neutral** sentiment. Supports both single-review
prediction and CSV batch prediction with visual analytics, built with a
Streamlit web interface.

## ✨ Features
- Single review sentiment prediction
- Batch CSV upload and prediction
- TF-IDF + Logistic Regression (primary model)
- Optional LSTM and BERT model implementations
- Summary statistics (counts & percentages per class)
- Visualizations: bar chart, pie chart, count plot, distribution graph, top words
- Downloadable prediction CSV

## 🛠 Technologies Used
- Python 3.10+
- NLTK (text preprocessing)
- Scikit-learn (TF-IDF, Logistic Regression, metrics)
- TensorFlow/Keras (LSTM, optional)
- HuggingFace Transformers + PyTorch (BERT, optional)
- Matplotlib & Seaborn (visualizations)
- Streamlit (GUI)
